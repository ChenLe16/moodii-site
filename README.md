# Moodii Site Template

This folder is a ready-to-copy template for a separate public GitHub Pages repo,
for example `moodii-site`.

## Files

- `index.html`
- `support.html`
- `privacy.html`
- `styles.css`

## Recommended setup

1. Create a new public GitHub repository, for example `moodii-site`.
2. Copy these files into the root of that repo.
3. Push to GitHub.
4. In the GitHub repo settings, enable GitHub Pages:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
5. Wait for GitHub Pages to publish.

Your URLs will then be similar to:

- `https://YOUR_USERNAME.github.io/moodii-site/`
- `https://YOUR_USERNAME.github.io/moodii-site/support.html`
- `https://YOUR_USERNAME.github.io/moodii-site/privacy.html`

## Before publishing

Replace these placeholders:

- support email if needed
- privacy policy effective date
- GitHub username / final site URL in App Store Connect
